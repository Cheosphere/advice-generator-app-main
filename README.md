# Aplicación generadora de consejos

El desafío es desarrollar esta aplicación generadora de consejos y lograr que se parezca lo más posible al diseño.

- HTML
- CSS
- JS Vanilla (with animejs library)

- Mobile: 375px
- Desktop: 1440px
